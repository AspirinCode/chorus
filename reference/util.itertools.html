Itertools utility¶

chunk(iterable, size)¶

chunk(‘ABCDEFG’, 3) –> ABC DEF G

chunk_truncate(iterable, size)¶

chunk_truncate(‘ABCDEFG’, 3) –> ABC DEF

chunk_fill(iterable, size, fillvalue=None)¶

chunk_fill(‘ABCDEFG’, 3, ‘x’) –> ABC DEF Gxx

consecutive(iterable, n)¶

consective(‘ABCDEF’, 3) –> ABC BCD CDE DEF consective(itertools.cycle(iter, n)) to get looped sequence
