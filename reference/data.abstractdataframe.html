Dataframe base class¶

class AbstractDataframe(df, attr_table)¶

Dataframe base class

  • Dataframe structure

      column
    name price ingredient
    attribute visible=True type=str visible=True type=int visible=True type=str
    row 0 sushi 1300 fish
    1 sukiyaki 1500 beef
    2 ramen 700 noodle

  • Get a record

    >>> df["price"][1]
1500

  • Get a column

    >>> [i for i in df["ingredient"]]
["fish", "beef", "noodle"]

  • Get a row

    >>> df.row(0)
["sushi", 1300, "fish"]

  • Iterate over rows

    >>> for row in df.rows_iter():
...     print(row)
["sushi", 1300, "fish"]
["sukiyaki", 1500, "beef"]
["ramen", 700, "noodle"]

  • Count rows

    >>> len(df)
3

  • Show column header

    >>> df.header()
["name", "price", "ingredient"]

  • Hide column

    >>> df["price"].visible = False
>>> df.header()
["name", "ingredient"]
>>> df.row(0)
["sushi", "fish"]
>>> for row in df.rows_iter():
...     print(row)
["sushi", "fish"]
["sukiyaki", "beef"]
["ramen", "noodle"]

  • Get and set column attribute

    >>> df["price"].type = int
>>> df["price"].type
int

    Phrase ‘apply’, ‘join’, ‘sort’ are preserved for method. These should not be used as attribute name.

  • join column

    >>> order = {"sushi": 2, "ramen": 1, "sukiyaki": 3}
>>> df["name"].join("num_of_order", order)
>>> df.header()
["name", "price", "ingredient", "num_of_order"]
>>> for row in df.rows_iter():
...     print(row)
["sushi", 1300, "fish", 2]
["sukiyaki", 1500, "beef", 3]
["ramen", 700, "noodle", 1]

  • apply function to the column

    >>> df["price"].apply("price_twice", lambda x: x * 2)
>>> df.header()
["name", "price", "ingredient", "price_twice"]
>>> for row in df.rows_iter():
...     print(row)
["sushi", 1300, "fish", 2600]
["sukiyaki", 1500, "beef", 3000]
["ramen", 700, "noodle", 1400]
