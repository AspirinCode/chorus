chorus.draw.helper¶

display_terminal_carbon(mol)¶

Set visible=True to the terminal carbon atoms.

equalize_terminal_double_bond(mol)¶

Show equalized double bond if it is connected to terminal atom.

spine_to_terminal_wedge(mol)¶

Arrange stereo wedge direction from spine to terminal atom

format_ring_double_bond(mol)¶

Set double bonds around the ring.

scale_and_center(mol)¶

Center and Scale molecule 2D coordinates. This method changes mol coordinates directly to center but not scale. This method returns width, height and MLB(median length of bond) and scaling will be done by drawer method with these values.

Returns:float height: float mlb: median length of bond
Return type:width
ready_to_draw(mol)¶

Shortcut function to prepare molecule to draw. Overwrite this function for customized appearance. It is recommended to clone the molecule before draw because all the methods above are destructive.
