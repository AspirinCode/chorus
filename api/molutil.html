chorus.molutil¶

clone(mol)¶

Returns deepcopy of the object. Pickle and unpickle is faster than deepcopy. TODO: this query function should be renamed to “cloned”

make_Hs_implicit(original_mol, keep_stereo=True)¶

Return molecule that explicit hydrogens removed TODO: this query function should be renamed to “explicitHs_removed”

largest_graph(mol)¶

Return a molecule which has largest graph in the compound Passing single molecule object will results as same as molutil.clone

mw(mol, ndigits=2)¶

Return standard molecular weight :param ndigits: number of digits

mw_wo_sw(mol, ndigits=2)¶

Molecular weight without salt and water :param ndigits: number of digits

charge(mol)¶

Total charge

non_hydrogen_count(mol)¶

Non-hydrogen atom count

H_donor_count(mol)¶

Hydrogen bond donor count

H_acceptor_count(mol)¶

Hydrogen bond acceptor count

rotatable_count(mol)¶

Rotatable bond count

rule_of_five_violation(mol)¶

Lipinski’s rule of five violation count

composition(mol)¶

Molecular composition in dict format (ex. Glucose {‘C’: 6, ‘H’: 12, ‘O’: 6}).

available_key(mol)¶

Eldest key +1

formula(mol)¶

Chemical formula. Atoms should be arranged in order of C, H and other atoms. Molecules should be arranged in order of length of formula text.
