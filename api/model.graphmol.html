add_molecule(mol, bond=None, base=None, target=None)¶

connect atom group (for SMILES parser)

May requires recalculation of 2D coordinate for drawing

Parameters:
  • mol – graphmol.Compound() the original object will be copied.
  • bond – Bond object to be connected. the original will not be copied so be careful.
  • base – index of atom in self to connect
  • target – index of atom in group to be connected
Raises:

TypeError
