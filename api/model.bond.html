chorus.model.bond¶

class Bond(attr_dict=None)¶

Bond object

  • Bond type
    • Single bond
      • 0: first - second
      • 1: first ◀ second (Up-arrow)
      • 2: first ◁ second (Down-arrow)
      • 3: first ~ second (Chiral)
    *Double bond
    • 0: second ニ first (clockwise, default)
    • 1: first ニ second (counter-clockwise)
    • 2: first ＝ second (equal length, for terminal bond by default)
    • 3: first × second (Cis-Trans Unknown)
Parameters:attr_dict (dict) – load Bond object from dict type notation.
order¶

int – bond order. smilesparser assigns order=9 to aromatic bonds.

is_lower_first¶

bool – bond direction. If true, lower index atom is the first atom and the other is the second atom. Bond.type attribute uses this feature for stereochemistry determination.

type¶

bond type (see above)

rotatable¶

bool – rotatable or not

aromatic¶

bool – aromatic or not

smiles_cis_trans¶

SMILES cis/trans flag

visible¶

bool – whether it should be drawn or not.
