
Chemex - Minimal set of chemical structure construction and drawing
====================================================================

Chemex is a simple Python module for chemical structure construction and drawing.


Features
----------

- Import molecule from V2000 Molfile or SMILES
- Draw molecule to SVG
- Draw molecule to PNG (requires PySide)
- Show molecular weight, formula, H-bond donor and acceptor
- Substructure search (VF2 algorithm in NetworkX)


Requirement
-------------

- Python 3.4 and upper
  - NetworkX
  - PyYAML


Optional
-------------

- PySide (PNG export)
