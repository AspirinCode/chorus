
Getting started
====================================================================


Installation
-------------



PyPI
^^^^^^

::

   pip3 install chorus



Anaconda
^^^^^^^^^

Distributing conda package is work in progress.

::

   conda upgrade -n root conda
   conda install -n root conda-build

   conda skeleton pypi chorus
   conda build chorus
   conda install --use-local chorus
